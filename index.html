Checked templating, gawk, sed options. Seemed that the best way to do this is to convert from Markdown to LaTeX at some point and then use one of the [Conditional compilation options](https://texfaq.org/FAQ-conditional) - [User Manual for Tagged package](https://mirror.niser.ac.in/ctan/macros/latex/contrib/tagging/tagging.pdf) ### MWE ~~~~~~ \documentclass{article} % sampleTagging.tex \usepackage{tagging} \usetag{author} %can use a list i.e {this,that} %\droptag{fancy} \begin{document} Hey this text is for all. \begin{taggedblock}{author} This is eyes only for author \end{taggedblock} \begin{taggedblock}{instructor} This is eyes only for instructor \end{taggedblock} \begin{taggedblock}{student} This is eyes only for student \end{taggedblock} ~~~~~~ ### Thru "Optional" Package ~~~~~~latex %One way to use this package is to declare (for example) \usepackage[opta]{optional} %at the beginning of your document, and flag optional text throughout your document like: \opt{opta}{Do this if option opta was declared} \opt{optb}{Do this if option optb was declared} \opt{optx,opty}{Do this if either option optx or opty} \opt{}{Never print this text!} \opt{opta}{\input{appendices}} \optv{xam}{Type: \verb|[root /]$ rm -r *|.} ~~~~~~ \end{document} About ============================================================== The logo-icon in the banner is from [Imgur](https://i.imgur.com/K4upfBe.png).
